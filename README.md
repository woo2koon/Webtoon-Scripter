# 🎨 Webtoon Script Manager (v2.0)

웹툰 대사 추출 및 배정 작업을 자동화하고 관리하는 전문가용 툴입니다. 구글 비전(Vision) API와 제미나이(Gemini) AI를 활용하여 번거로운 수작업을 획기적으로 줄여줍니다.

---

## 🚀 핵심 기능

### 1. 스마트 OCR (텍스트 추출)
*   **스마트 모드:** 웹툰의 흰색 여백(컷 사이)을 지능적으로 분석하여 이미지를 자릅니다. 대사가 잘리는 현상 없이 완벽하게 텍스트를 추출합니다.
*   **말풍선 그룹화:** 흩어져 있는 텍스트 블록을 하나의 말풍선 단위로 자동 병합하여 자연스러운 대사 흐름을 복원합니다.
*   **캐싱 시스템:** 한 번 분석한 이미지는 결과값을 저장하여 중복 API 비용을 절감합니다.

### 2. AI 맞춤법 검사
*   **Gemini AI 연동:** 구글의 최신 AI 모델을 사용하여 문맥에 맞는 한국어 맞춤법 및 띄어쓰기를 교정합니다.
*   **원형 보존:** 웹툰 특유의 어조와 줄바꿈 구조를 유지하면서 문법적 오류만 정확하게 수정합니다.

### 3. 유연한 작업 모드
*   **심플 모드 (Simple Mode):** 복잡한 설정 없이 이미지를 드래그 앤 드롭하여 즉시 텍스트를 추출하고 교정할 수 있는 쾌속 모드입니다.
*   **프로젝트 모드 (Full Mode):** 작품별, 회차별로 데이터를 체계적으로 관리합니다. 캐릭터 배정, 엑셀 내보내기 등 전체 공정을 지원합니다.

### 4. 캐릭터 및 스크립트 관리
*   **캐릭터 DB:** 등장인물의 이름, 역할, 연령, 성별 등을 설정하고 관리할 수 있습니다.
*   **스프레드시트 에디터:** 엑셀과 유사한 인터페이스로 대사별 캐릭터를 직관적으로 배정합니다.
*   **데이터 내보내기:** 작업 완료 후 엑셀(.csv) 또는 텍스트(.txt) 파일로 즉시 저장하여 후속 작업에 활용할 수 있습니다.

### 5. 고성능 뷰어
*   **최적화된 스크롤:** 아주 긴 웹툰 이미지도 끊김 없이 부드럽게 볼 수 있도록 설계되었습니다.
*   **다크/라이트 배경:** 작업 환경에 맞춰 뷰어의 배경색을 조정하여 가독성을 높일 수 있습니다.

## 🛠 기술 스택
*   **Language:** Python 3.10+
*   **UI Framework:** PySide6 (Qt for Python)
*   **AI/OCR:** Google Cloud Vision API, Google Gemini AI
*   **Data:** Pandas, JSON

---

## 📦 설치 및 실행 방법

1.  **의존성 설치**
    ```bash
    pip install -r requirements.txt
    ```

2.  **프로그램 실행**
    ```bash
    python main.py
    ```

3.  **API 키 설정**
    *   `파일 > 설정` 메뉴에서 Google Cloud API 키와 Gemini API 키를 입력해야 모든 기능을 사용할 수 있습니다.
